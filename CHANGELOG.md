# 更新日志

所有对 CF-Ares 的显著更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，

## [0.1.0] - 2024-03-04

### 新增

- 初始版本发布
- 支持使用 SeleniumBase 和 undetected-chromedriver 突破 Cloudflare 防护
- 支持使用 curl_cffi 进行高性能请求
- 支持浏览器指纹管理
- 支持会话管理和复用
- 提供类似 requests 的 API 接口
- 添加基本示例和高级示例
